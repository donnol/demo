# WebAssembly

[WebAssembly 是便携式的抽象语法树，被设计来提供比 JavaScript 更快速的编译及运行。WebAssembly 将让开发者能运用自己熟悉的编程语言（最初以 C/C++作为实现目标）编译，再藉虚拟机引擎在浏览器内运行](https://zh.wikipedia.org/wiki/WebAssembly)

    传输的时候是二进制格式(wasm)，阅读的时候是文本格式(wast)
